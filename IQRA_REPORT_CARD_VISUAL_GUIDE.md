## Print Behavior

### When Print Button Clicked
```
1. Hide screen controls
2. Show print container
3. Render report card(s)
4. Trigger browser print dialog
5. After print: restore screen view
```

### Print All Students
```
For each student:
  - Generate report card
  - Add page break
  - Continue to next student
  
Result: Multi-page PDF with all students
```
